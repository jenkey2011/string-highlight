# string-highlight

Syntax highlight for code in string. Such as html tags in template literals.

![](src/static/diff.png)

![](src/static/show.gif)

## Features

- supprt full html tags
- support `.js`、`.jsx`、`.ts`、`.tsx`
- support emmt
